--------------------------------------------
APPLICATION 1 — ENERGY CONSUMPTION
--------------------------------------------
Title: "Cálculo do Consumo Energético"

Mathematical model:
E = ∫[t₁,t₂] P(t) dt

Where:
- P(t): Power function in watts (W)
- t: time (hours)
- E: energy in watt-hours (Wh)

User inputs:
- Power function P(t) (example: 100 + 20*t)
- Initial time t₁
- Final time t₂

Outputs:
- Numerical value of E
- Graph of P(t)
- Textual interpretation:
  "A energia total consumida neste intervalo é X Wh,
   equivalente a aproximadamente Y kWh."

--------------------------------------------
APPLICATION 2 — NETWORK DATA TRANSFER
--------------------------------------------
Title: "Cálculo de Dados Transferidos numa Rede"

Mathematical model:
D = ∫[t₁,t₂] R(t) dt

Where:
- R(t): data rate in Mbps
- t: time in seconds
- D: total data transferred (Megabits)

User inputs:
- Data rate function R(t) (example: 50 + 10*sin(t))
- Initial time t₁
- Final time t₂

Outputs:
- Total data transferred in Megabits
- Conversion to Megabytes (MB)
- Graph of R(t)
- Textual interpretation:
  "Durante este período, foram transferidos X MB de dados."

--------------------------------------------
APPLICATION 3 — SERVER COMPUTATIONAL LOAD
--------------------------------------------
Title: "Carga Computacional de um Servidor"

Mathematical model:
C = ∫[t₁,t₂] CPU(t) dt

Where:
- CPU(t): CPU usage percentage (0–100)
- t: time in seconds
- C: cumulative computational load

User inputs:
- CPU usage function CPU(t) (example: 30 + 40*t/(t+10))
- Initial time t₁
- Final time t₂

Outputs:
- Numerical value of C
- Graph of CPU(t)
- Interpretation:
  "Este valor representa a carga computacional acumulada,
   sendo útil para análise de desempenho e planeamento de recursos."
